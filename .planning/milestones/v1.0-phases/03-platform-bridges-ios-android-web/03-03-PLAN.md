    - Get activity from `activityProvider()` -- if null, callback with `FlutterError("notSupported", "No activity available", null)`

    - Create `CreatePublicKeyCredentialRequest(requestJson = optionsJson)`
    - Launch coroutine in `scope`:
      ```kotlin

              val response = result as CreatePublicKeyCredentialResponse

              callback(Result.failure(FlutterError("cancelled", e.message ?: "User cancelled", null)))

      }
      ```
